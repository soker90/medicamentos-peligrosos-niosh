📊 Fuente de Datos

Esta herramienta utiliza datos oficiales del NIOSH List of Hazardous Drugs in Healthcare Settings, 2024.

  • Documento oficial: NIOSH 2025-103 PDF ↗
  • Organismo: CDC/NIOSH (Centers for Disease Control and Prevention)
  • Total de medicamentos: 130 (46 Grupo 2, 84 Grupo 3)
  • Versión del documento: 2024 (publicado 2025)
